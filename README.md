## 除了用到四大组件外还用到的一些小组件
### ImageLoader
这是我在github上看到的一个别人写的组件：
- 它可以用来加载图片
- 同时它也可以用来加载网络的图片
- 如果加载的图片来自网络，那么它会将该图片存到一个缓存区中。当再次用到这张图片的时候，就会在缓存区中取，而不是从网络上获取
### PhotoView
这也是我在github上看到的一个别人写的组件
- 它可以使图片放大和缩小
